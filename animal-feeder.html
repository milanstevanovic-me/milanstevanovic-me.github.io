Arduino Automated Animal Feeder

A self-built, programmable feeder with servo-controlled
dispensing and real-time feedback signals.

← Back to Projects

Overview

This was an independent project built to solve a real problem: automatically dispensing food for a pet on a reliable schedule, without needing to be home. I designed and built the entire system myself — from the mechanical dispensing mechanism to the Arduino code controlling it.

How It Works

The feeder uses a servo motor to rotate a dispensing mechanism at programmed intervals, releasing a controlled portion of food each cycle. An Arduino microcontroller handles the timing logic, and the system gives feedback through an LED and buzzer — confirming when a feeding cycle has successfully completed, and flagging if something goes wrong.

Design & Build Process

  • Designed the mechanical housing and dispensing mechanism to reliably portion food without jamming
  • Selected and integrated a servo motor for precise, repeatable rotation control
  • Programmed the Arduino to handle scheduled timing and feeding intervals
  • Added LED and buzzer feedback to confirm successful dispensing cycles
  • Tested and iterated on the mechanism to prevent food jams and ensure consistent portion sizes

Key Takeaways

This project gave me hands-on experience combining mechanical design with embedded electronics — an area I'm increasingly interested in (mechatronics). It also taught me a lot about designing for reliability in a system that has to run unattended.
